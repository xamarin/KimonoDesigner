#Android Test Consumer

####Version 01.01.00

##Summary

Kimono Designer allows a developer to graphically edit Skia based graphic objects that can be converted to code and included in any cross-platform language and OS that SkiaSharp supports. This project is used to test the source code generated by Kimono Designer for the Android OS.
